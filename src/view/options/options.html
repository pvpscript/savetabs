How to use

Window options

Choose between two types of window selection.

  1. Current window

    2. Highlight every tab for the current window.

  2. Every window

    3. Highlight every tab for every opened window.

Format options

Change between different output formats and its sub-options.

  1. Plain text

    2. Output file saved as *.txt plain text format.

    1. Raw copy -raw

      2. Hide window header.

  2. JSON

    3. Output file saved in *.json format.

    1. Minified -minified

      2. Properties grouped without any spacing.

    2. Readable -readable

      3. Properties grouped readably, with spaces and tabs.

Buttons

Choose a way to save the file. The saving name is automatically generated using the current date time + a suffix.

  1. Save As

    2. Prompt a window to choose a saving location and name.

  2. Quicksave

    3. Saves automatically in the default/last download location.
